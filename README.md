# roblox-vector-search

This is a TypeScript project using pnpm. Source files are in `src/` and compiled output is in `dist/`.

## Setup

1. Install dependencies:
   ```sh
   pnpm install
   ```
2. Build the project:
   ```sh
   pnpm run build
   ```
3. Run the project:
   ```sh
   pnpm start
   ```

## Project Structure

- `src/` - TypeScript source files
- `dist/` - Compiled JavaScript output
